# Configuración

Esta sección describe la configuración mínima para ejecutar la API.

## Variables de entorno

- `ASPNETCORE_ENVIRONMENT`: Entorno de ejecución (`Development`, `Production`).
- `ConnectionStrings__DefaultConnection`: Cadena de conexión a SQL Server.
- `API_KEY_SECRET`: Clave secreta para validar API Keys.

## URL base de la API

```text
https://localhost:5001/api/transportista
```
